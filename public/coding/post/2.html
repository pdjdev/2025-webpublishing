Python으로 데이터 분석하기 - Pandas와 Matplotlib 완벽 가이드
작성일: 2025년 5월 11일 · 작성자: 박동준

데이터는 21세기의 석유라고 불립니다. 하지만 원유를 정제해야 사용할 수 있듯이, 데이터도 분석과 시각화를 통해 의미 있는 인사이트로 변환되어야 합니다. Python은 이러한 데이터 분석 작업에 가장 적합한 언어 중 하나이며, Pandas와 Matplotlib은 이 과정에서 핵심적인 도구입니다.

이번 글에서는 실제 데이터를 사용하여 Pandas와 Matplotlib의 기본기부터 고급 활용법까지 단계별로 알아보겠습니다.

1. 환경 설정 및 라이브러리 설치

먼저 필요한 라이브러리들을 설치해보겠습니다. 터미널이나 명령 프롬프트에서 다음 명령어를 실행하세요:

pip install pandas matplotlib seaborn numpy jupyter

2. Pandas 기초 - 데이터 로딩과 탐색

Pandas는 데이터 조작과 분석을 위한 강력한 라이브러리입니다. CSV, Excel, JSON 등 다양한 형식의 데이터를 쉽게 불러올 수 있습니다.

import pandas as pd
import matplotlib.pyplot as plt
import numpy as np

# CSV 파일 읽기
df = pd.read_csv('sales_data.csv')

# 데이터 기본 정보 확인
print(df.head())      # 처음 5행
print(df.info())      # 데이터 타입과 null 값 정보
print(df.describe())  # 기본 통계 정보

주요 Pandas 함수들

  • df.shape: 데이터프레임의 크기 (행, 열)
  • df.columns: 컬럼명 리스트
  • df.isnull().sum(): 각 컬럼의 결측값 개수
  • df.value_counts(): 각 값의 빈도수

3. 데이터 정제와 변환

실제 데이터는 항상 깔끔하지 않습니다. 결측값 처리, 중복 제거, 데이터 타입 변환 등의 정제 과정이 필요합니다.

# 결측값 처리
df_cleaned = df.dropna()  # 결측값이 있는 행 제거
df_filled = df.fillna(df.mean())  # 평균값으로 채우기

# 중복 제거
df_unique = df.drop_duplicates()

# 데이터 타입 변환
df['date'] = pd.to_datetime(df['date'])
df['category'] = df['category'].astype('category')

# 새로운 컬럼 생성
df['total_sales'] = df['quantity'] * df['price']
df['month'] = df['date'].dt.month

4. Matplotlib으로 데이터 시각화

데이터를 이해하는 가장 효과적인 방법 중 하나는 시각화입니다. Matplotlib은 Python에서 가장 기본적이면서도 강력한 시각화 라이브러리입니다.

기본 차트 그리기

# 선 그래프
plt.figure(figsize=(10, 6))
plt.plot(df['date'], df['sales'], marker='o')
plt.title('일별 매출 추이')
plt.xlabel('날짜')
plt.ylabel('매출액')
plt.xticks(rotation=45)
plt.tight_layout()
plt.show()

# 막대 그래프
category_sales = df.groupby('category')['sales'].sum()
plt.figure(figsize=(8, 6))
category_sales.plot(kind='bar')
plt.title('카테고리별 총 매출')
plt.xlabel('카테고리')
plt.ylabel('매출액')
plt.show()

# 히스토그램
plt.figure(figsize=(8, 6))
plt.hist(df['sales'], bins=20, alpha=0.7, color='skyblue')
plt.title('매출 분포')
plt.xlabel('매출액')
plt.ylabel('빈도')
plt.show()

5. 고급 분석과 시각화

기본적인 차트를 넘어서 더 정교한 분석과 시각화를 해보겠습니다.

상관관계 분석

# 상관관계 매트릭스
correlation_matrix = df[['price', 'quantity', 'sales']].corr()
plt.figure(figsize=(8, 6))
plt.imshow(correlation_matrix, cmap='coolwarm', aspect='auto')
plt.colorbar()
plt.title('변수간 상관관계')
plt.xticks(range(len(correlation_matrix.columns)), correlation_matrix.columns)
plt.yticks(range(len(correlation_matrix.columns)), correlation_matrix.columns)
plt.show()

시계열 분석

# 월별 매출 트렌드
monthly_sales = df.groupby(df['date'].dt.to_period('M'))['sales'].sum()
plt.figure(figsize=(12, 6))
monthly_sales.plot(kind='line', marker='o')
plt.title('월별 매출 트렌드')
plt.xlabel('월')
plt.ylabel('매출액')
plt.grid(True, alpha=0.3)
plt.show()

6. 실전 팁과 모범 사례

성능 최적화

  • 대용량 데이터: chunksize 매개변수를 사용하여 데이터를 청크 단위로 처리
  • 메모리 효율성: 적절한 데이터 타입 사용 (int8, float32 등)
  • 벡터화 연산: 반복문 대신 Pandas의 벡터화된 연산 활용

시각화 개선 팁

  • 색상 일관성: 브랜드 색상이나 의미 있는 색상 팔레트 사용
  • 폰트 크기: 가독성을 위한 적절한 폰트 크기 설정
  • 범례와 라벨: 명확하고 이해하기 쉬운 라벨링

결론

Pandas와 Matplotlib은 데이터 분석의 필수 도구입니다. 이 두 라이브러리를 잘 활용하면 복잡한 데이터에서 의미 있는 인사이트를 추출하고, 효과적으로 시각화할 수 있습니다.

가장 중요한 것은 꾸준한 실습입니다. 실제 데이터를 가지고 직접 분석해보면서 경험을 쌓아나가시기 바랍니다. 데이터 분석은 기술적 스킬뿐만 아니라 도메인 지식과 비즈니스 이해도 중요한 역할을 한다는 점도 기억해주세요.
